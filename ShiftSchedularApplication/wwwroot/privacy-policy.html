ShiftSchedularApplication Privacy Policy

About This Application

This is a student project for educational purposes developed as part of the JavaScript Frameworks course.

Data Collection and Usage

We do not store or share user data beyond what is necessary for the application's core functionality:

  • User authentication information (managed by ASP.NET Core Identity)
  • Shift scheduling data (employee shifts, availability, time punches, pay stubs)
  • Basic user profile information for application functionality

Third-Party Authentication

This application supports authentication through:

  • Google OAuth - for user login and account creation
  • Facebook OAuth - for user login and account creation

When using these services, you are subject to their respective privacy policies.

Data Security

All data is stored securely using industry-standard encryption and security practices. We do not sell, trade, or otherwise transfer your personal information to third parties.

Contact Information

This is an educational project. For questions about this application, please contact your course instructor.
